### Обо мне

Системный аналитик с техническим бэкграундом и опытом автоматизации бизнес-процессов.
Работаю на стыке бизнеса, разработки и данных: собираю и формализую требования,
проектирую API, описываю процессы, модели данных и сценарии взаимодействия систем.

Есть опыт полного цикла: от анализа текущих процессов и проектирования решения
до сопровождения разработки, тестирования, внедрения и обучения пользователей.

Основной фокус сейчас — системная аналитика, поиск команды для участия в новых интересных проектах.

### Основной фокус

- Сбор, анализ и формализация требований
- Функциональные и нефункциональные требования
- Проектирование REST API и интеграций
- Моделирование бизнес-процессов и пользовательских сценариев
- Моделирование данных и описание структуры БД
- Сопровождение разработки, тестирования и внедрения
